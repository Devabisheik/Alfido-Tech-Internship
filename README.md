# ALFIDO TECH INTERN PROJECTS(TASKS) 

## Overview

This repository contains three Java projects developed as part of my learning and internship tasks. Each project focuses on different Java concepts such as Object-Oriented Programming, JDBC, Database Management, Collections Framework, and software design principles.

The projects are developed using Java and demonstrate how real-world applications can be implemented through console-based systems.

---

## Projects Included

### Task 1 - Online Food Ordering System

A console-based food ordering application that allows customers to browse menus, place food orders, make payments, and track delivery status.

**Concepts Used:**

* Object-Oriented Programming
* Encapsulation
* Inheritance
* Composition
* Aggregation
* Collections Framework
* Enums

---

### Task 2 - Banking Management System

A banking application developed using Java, JDBC, and MySQL. The system manages customer accounts, transactions, deposits, withdrawals, fund transfers, and transaction history.

**Concepts Used:**

* JDBC Connectivity
* MySQL Database
* SQL Queries
* Exception Handling
* Database Relationships
* Object-Oriented Programming

---

### Task 3 - E-Commerce Order Management System

A console-based e-commerce application that manages products, customers, orders, payment processing, and order status tracking.

**Concepts Used:**

* Object-Oriented Programming
* Interfaces and Abstraction
* Collections Framework
* HashMap Storage
* Exception Handling
* Order Lifecycle Management
